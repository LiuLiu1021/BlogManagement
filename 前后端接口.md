# method没有写的默认就是get

## 返回的data中全都有一个code, 'success'表示成功， ‘error’表示不成功



# 前后端接口

## 1. index

**getNewBlog()**

```json
url:'/index/getNewBlog',
method: 'get',
params:{
    currentPage:1,//当前页
    pageSize:10//每页大小,即每次返回第( (currentPage-1)*pageSize,currentPage*pageSize)个博客，按时间排序
}
data:{
    total:100,//博客总数量
    size:10,//单页page博客数量
    currentPage:1//现在位于第几页
    records:[
{
    title: '标题',
 	author: '作者',
 	labels: ['标签','标签'] 
 	summary: '概括', 
 	clickCount: 100,
 	likeCount: 200,
 	time: '2020-12-2'
}，
{
    title: '标题',
 	author: '作者',
 	label: '标签', 
 	summary: '概括', 
 	clickCount: 100,
 	likeCount: 200,
 	time: '2020-12-2'
}]
}
```
